# 🐟 水滴魚桌寵 (Blobfish Desktop Pet)

一隻住在 macOS 桌面底部的水滴魚,會自己游來游去、會被揍、會提醒你該下班了。

用 Electron 打造,全手繪 SVG,無外部素材依賴。

## 功能

### 互動
- **自由游動** — 在螢幕底部水平來回,碰到邊緣自動轉向
- **點擊揍牠** — 身體擠壓變形、冒出眼淚,並隨機回一句嘴(8 種反應)
- **拖曳移動** — 拖到螢幕任何位置,放開後就在該高度繼續游動
- **甩飛物理** — 用力一甩會帶著慣性飛出去,撞到螢幕邊緣會 Q 彈反彈,摩擦力遞減後停下並恢復游動
- **隨機碎念** — 每 1~3 分鐘冒出一句閒聊(34 種語料)

### 工作提醒
| 時間 | 提醒內容 |
|------|----------|
| 每個整點 / 30 分 | 主人您辛苦了,您又工作了半小時。 |
| 12:55 | 主人還有五分鐘就可以去吃飯啦,您辛苦啦。 |
| 18:30 | 主人還有半個鐘就下班啦,主人今天太棒啦! |
| 18:55 | 主人還有五分鐘就下班啦...(週五自動改成「下週見」) |

### 其他
- 永遠置頂,跨所有桌面空間顯示
- 滑鼠穿透 — 只有魚身會擋住點擊,其餘區域完全不影響操作
- 多螢幕支援 — 可拖曳至任何已連接的螢幕
- 選單列 🐟 圖示,可隨時結束程式

## 開發

```bash
npm install
npm start
```

## 打包

```bash
npm run dist
```

產出:
- `dist/mac/水滴魚.app` — 可直接雙擊執行,或拖到「應用程式」資料夾
- `dist/水滴魚-1.0.0-mac.zip` — 分享用壓縮檔

> **注意**:App 未經 Apple 簽名。在其他 Mac 首次開啟時,需到「系統設定 → 隱私權與安全性」點選「仍要打開」。
>
> 打包目標為 Intel (x64),Apple Silicon 可透過 Rosetta 執行。

## 開機自動啟動

將「水滴魚.app」拖入「系統設定 → 一般 → 登入項目」。

## 技術細節

| 檔案 | 職責 |
|------|------|
| `main.js` | 主程序:視窗管理、游動/甩飛物理運算、提醒排程、選單列 |
| `preload.js` | IPC 橋接層(context isolation) |
| `renderer.js` | 渲染層:拖曳偵測、動畫觸發、對話泡泡、語料庫 |
| `index.html` | SVG 造型與 CSS 動畫 |

### 一些踩過的坑

- **`Math.round(-0.5)` 回傳 `-0`**,Electron 的 `setPosition` 原生綁定會直接拒絕負零並丟出例外。魚往左游經過螢幕原點時必炸,所有座標都需經 `|| 0` 正規化。
- **macOS 會強制把視窗擋在選單列下方**,無視程式設定的座標(連 `type: 'panel'` 也繞不過)。邊界計算必須用 `workArea` 而非 `bounds`,否則內部追蹤的位置會與系統實際位置脫節。
- **視窗自己移動時不會觸發 `mousemove`**,滑鼠穿透狀態因此不會更新。需由主程序在移動後主動將游標座標傳給渲染層重新判定。
- **CSS class 忘記移除會靜默蓋掉後續動畫** — `bump` 與 `hit` 優先度相同時,後宣告者勝出。
